Transmittals

The following section details the process for you to create and review transmittals.

Create Transmittal File

To create a transmittal file, perform the following steps:

1.   From the Navigation Menu, select Transmit.

2.   Click the arrow in the Originals card to view the Setup Transmit sliding panel.

Setup Transmit

From the Setup Transmit sliding panel, you can configure a transmittal to include specified payers, form types, and records under the Data Selection section.

1.   To select which payers you want to include in the transmittal, perform the following steps:
a. Click the Edit button associated with Payers.

b.   Check the box associated with the payer(s) you want to include. Forms associated with the payer(s) you’ve selected will be included in the transmittal. By default, all payers are checked.

Note: The numbers within the parenthesis indicate the number of payers that you’ve selected out of the number of Payers available.

2.   To select which forms you want to include in the transmittal, perform the following steps:
a. Click the Edit button associated with Forms.

b.   Check the box associated with the form(s) you want to include. All forms for the payer(s) you have selected will be included in the transmittal. By default, all forms are checked.

Note: The numbers within the parenthesis indicate the number of forms that you’ve selected out of the number of forms available.

3.   If you want to include forms under the federal limit, check the Include Forms Under Federal Limit box. If you leave this box unchecked, forms under the federal limit will be excluded from the transmittal.

4.   If you want to include forms with errors, check the Include Forms with Errors box. When you check this box, any form whose recipient has errors will also be included. If you leave this box unchecked, forms with errors will be excluded from the transmittal.

5.   If you want to send the transmittal to the IRS, along with any CFS states, check the Combine Federal and State (CFS) box. If you leave this box unchecked, federal and state limits will be applied and the Include Forms Under Federal Limit check box will be disabled.

6.   Control totals are displayed for each form type. If you want to view the control totals, click the arrow associated with the relevant form. The table displayed will provide the control totals of the records being included based on the criteria you’ve selected under Data Selection.

7.   If the Refresh Data button is enabled, the control totals displayed are not the most recent. If you want to pull the latest control totals, click this button. The Refresh Data button will be enabled when you have altered at least one item under Data Selection.

Note: If the Refresh Data button is enabled and you’ve clicked the Review button, the control totals produced in the print job may not match the information displayed in the Setup Transmit sliding panel.

8.   Once you have set up all of the desired settings and the transmittal is ready to be generated, click Review.

Review or Cancel Transmittal

Once you’ve clicked Review, the transmittal file will begin to generate. As the transmittal file is generating, the Review Transmit sliding panel will be displayed with a read-only view of the Data Selection settings you’ve selected.

1.   A spinner in the center of the Review Transmit sliding panel will indicate that the transmittal is still in progress.

2.   Once the spinner disappears and the “Transmittal generated” snackbar message has appeared, the transmittal file has been generated, but not uploaded to the IRS.

3.   You can then expand the control totals for the form(s) you’ve selected.

4.   To cancel the transmittal, click Cancel Transmittal. From the confirmation pop-up window, click Cancel Transmit for you to cancel and delete the transmittal. The forms included in the transmittal you’re canceling will be made available for you transmit when you want.

5.   When you click out of the Review Transmit sliding panel, or click the X on the upper right-hand corner, the transmittal will not be canceled. The transmittal will be saved for you to approve at a later time. The Transmittals section of the Transmit page will display an Awaiting Approval card; click the associated arrow to open the transmittal.

Approve and Submit Transmittal

To approve and submit the transmittal, perform the following steps:

1.   From the Review Transmit sliding panel, click Transmit.

2.   A pop-up confirmation message will appear, providing the options for you to either Go Back or Transmit to IRS.

3.   You can track the status of the transmittal on the Transmit page.
