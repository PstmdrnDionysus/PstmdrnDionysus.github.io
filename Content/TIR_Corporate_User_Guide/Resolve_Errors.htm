Resolve Errors

Errors can be resolved from the Overview, Data, and Import Results Summary pages.

To resolve errors, perform the following steps:

1.   Click the Errors card from either the Overview or Data page.
a. Overview page Errors card:

b.   Data page Errors card:

c.   Import Results Summary page Errors card:

Note: The only difference on the Import Results Summary page is that the Errors card represents the number of errors in the import that was just performed.

2.   A list of all Errors will be displayed in the given context, as well as the number of records each error has.

3.   You can click each error to view a list of records that have the error you’ve selected.

4.   You can also click each record to view the associated Data Entry sliding panel.

5.   Once an error is resolved, each sliding panel will be updated with the current status of the product. For example, if there is one record with an invalid state value and the error has been resolved, that error will be removed from the list.
