Data Reset

The following section details the process for resetting data.

To delete data by record type, perform the following steps:

1.   Click the Settings icon, .

2.   From the Data Reset card, click Select Data To Reset.

3.   Rules for resetting data are as follows:
a. You cannot reset data if at least one form has been printed or transmitted in that context.
b.   You cannot reset recipient and payer data if there are any tax detail forms within the system.
c.   You can reset recipient and payer data if all tax forms are reset at the same time.
4.   Select the record type(s) you want to reset, enter the Code, and click Reset Data.
