User Provisioning

The following section details the user provisioning process.

1.   From the Navigation Menu, click the Settings icon, .

2.   From the Settings page, click Manage Organizations under User.

3.   From this page, you can create, delete, or edit users.

Create User

To create a new user, perform the following steps:

1.   From the Users Management page, click New User.

2.   From the User Info tab, under Basic Settings, enter the user’s Email, First Name, and Last Name; all other fields should be left blank or as defaulted.

3.   Navigate to the Roles tab and select the type of role you want to associated with that user using the associated TIR Corporate drop-down options.

Role Options

TIR_Admin

• User management.
• Data entry/import.
• Create Print review of individual record.
• Create/approve Print files.
• Create/approve Transmittal files.
• Upload/download files.
• TIN Matching.

TIR_EDITOR

• Data entry/import.
• Create Print preview of individual record.
• Create Print files.
• Create Transmittal files.
• Upload/download files.
• TIN Matching.

TIR_READ_ONLY

• View data.
• Create Print preview of individual record.
4.   Click Save; an email will automatically be sent to the user’s email address from no-reply-platform@sovos.com, which will lead the user through an intuitive activation process.

Edit User

To edit a user, perform the following steps:

1.   From the Users Management page, click the user you want to edit.

2.   Update the necessary information and click Save.

Delete User

To delete a user, perform the following steps:

1.   Check the box associated with the user(s) you want to delete and click the icon, which appears after selecting a user(s).

2.   From the confirmation pop-up message, click Delete.
