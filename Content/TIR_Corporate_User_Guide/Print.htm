Print

The following sections detail the print processes available in TIR - Corporate.

Create Print File

To create a print file, perform the following steps:

1.   From the Navigation Menu, select Print.

2.   Click the arrow within the Not Started card to view the Setup Print sliding panel.

Setup Print

The Setup Print sliding panel allows you to configure the Data Selection and Print Options. Data Selection includes selecting Payers and Forms.

 

Data Selection

1.   To select which payers you want to include in the print job, perform the following steps:
a. Click the Edit button associated with Payers.

b.   Check the box associated with the desired payer(s). Forms associated with the payer(s) you’ve selected will be included in the print job. By default, all payers are checked.

Note: The numbers within the parenthesis indicate the number of payers that you’ve selected out of the number of payers available.

2.   To select which forms you want to include in the print job, perform the following steps:
a. Click the Edit button associated with Forms.

b.   Check the box associated with the desired form(s). All forms for the payer(s) you’ve selected will be included in the print job. By default, all forms are checked.

Note: The numbers within the parenthesis indicate the number of forms you’ve selected out of the number of forms available.

3.   Check the Include Forms Under Federal Limit box to include forms under the federal limit in the print job. If you leave this box unchecked, forms under the federal limit will be excluded from the print job.

4.   Check the Include Forms with Errors box to include forms with errors. When you check this box, any form whose recipient has errors will be included. If you leave this box unchecked, forms with errors will be excluded from the print job.

Print Options

1.   Under the Print Options section, you can add Print Notes, turn on or off a Mask TIN, and you can chose to have the print job can be chosen to be printed with Sovos Print Services, if desired.

Note: For more information on Sovos Print Services, contact your Account Manager.

2.   To add a Print Note, click the associated Edit button.

3.   From the Add Print Notes drop-down list, click New Print Note.

4.   Fill in the required Add Print Note Name field and enter Print Notes into the four Print Lines below; each line allows up to 34 characters.

5.   To save the Print Notes, click Save. If you want to permanently delete the Print Notes, click Delete.

Note: IA Print Note that you’ve created for a 1099-DIV can be applied to a 1099-INT and/or 1099-MISC, and vice versa, Follow the same process to add a Print Note and the previously created Print Note will appear in the drop-down list

6.   To view which Print Note you want to apply to each form, select the previously created Print Note. from the associated drop-down list.

7.   Click the arrow associated with the relevant form to expand the information. Once you’ve edited all Print Lines as needed, click Save Edits.

8.   From the Setup Print sliding panel, the numbers associated with the Print Notes will indicate the number of forms with print notes.

9.   To mask or unmask TINs on the print file, click the Edit button associated with Mask TINs on Forms.

10.   By default, TIN masking is enabled for all forms. You can turn TIN Masking on or off by form type. When you check the box associated with a form, TIN Masking for that form will be enabled. When TIN Masking is enabled, TINs will be printed, displaying only the last four numbers of the TIN.

11.   If Sovos Print Services has been purchased, the Use Sovos Print Services box will be visible. Check this box to send the generated print file to Sovos to be printed. When you leave this box unchecked, the generated print file will not be sent to Sovos to be printed. Instead, a PDF will be generated to be printed locally.

Note: Under Recipients, a Print Locally check box is available for you to use. When you check the Use Sovos print services box, the information included will only be recipients who have Print Locally unchecked in the print job, along with their associated forms. When you leave the Use Sovos print services box unchecked, only recipients who have Print Locally checked, along with their forms, will be included in the print job.

Control Totals

1.   Control totals are displayed for each form type. To view the control totals, click the arrow associated with the relevant form. The table displayed will provide the control totals of the records being included based on the criteria you’ve selected from the Data Selection and Use Sovos print services options.

2.   If the Refresh Data button, located under the Include Forms with Errors check box, is enabled, the Control Totals displayed are not the most recent. Click this button to pull the latest Control Totals. The Refresh Data button will be enabled when you’ve altered at least one item in the Data Selection, or when you’ve updated the Use Sovos print services.

Note: If the Refresh Data button is enabled and you’ve clicked the Review button, the control totals produced in the print job may not match the information displayed in the Setup Print sliding panel.

3.   Once you’ve set all settings and you’re ready to generate the print job, click Review.

Review or Cancel Print

Once you’ve clicked Review, the print file will begin to generate. As the print file is generating, the Review Print sliding panel will be displayed with a read-only view of the Data Selection and Print Options settings you’ve selected.

1.   The top of the page will display a tab for each form type being generated. A spinning icon associated with the name of the form type will indicate that the print job is still in progress.

2.   Once the spinner turns to a check mark icon, and the “Print generated” snackbar message has appeared, the print file has been generated.

3.   Expand the Control Totals for the form(s) you’ve selected; a preview of all forms being printed will be displayed in the center of the page.

4.   The print Preview allows you to view each individual form prior to approving the print job. If you want to view a different form, perform the following steps:
a. Use the arrows located at the top of the print Preview to scroll to the next form.

b.   Use the Search box in the upper right-hand corner of the print Preview to search for a form by the name, TIN, or account number of the recipient.

5.   Click the arrow associated with the form’s Control Totals to view the control totals broken down by each box and payer.

6.   If you want to cancel the print job, click Cancel Print; a confirmation pop-up window will appear. Click Cancel Print from this window to cancel and delete the print job. The forms included in the Print job being canceled will be made available again to be printed.

7.   When you click out of the Review Print sliding panel, or click the X on the upper right-hand corner of the sliding panel, the print job will not be canceled. Instead, the print job will be saved to be approved at a later time. The Prints section of the Print page will display an Awaiting Approval card. click the associated arrow to open the print job.

Approve Print Job

To approve the print job being printed with Sovos, perform the following steps:

1.   From the Review Print sliding panel, click Print to Sovos in the upper right-hand corner of the page.

2.   A pop-up confirmation message will appear, providing the option for you to do any of the following three actions:
a. Download a PDF of each form type.

b.   Cancel the print order. You will be brought back to the Review Print sliding panel.

c.   Once you’ve clicked Print To Sovos, the process of the print file being printed and mailed can no longer be canceled.

3.   Once the print job is Printed to Sovos, the Print Order Status tracker will appear on the Print page. You can use this to track each form type in the print job individually.

4.   When you click into each individual form type within the Print Order Status tracker, you can view a sliding panel containing the control totals, along with a general summary of the print file.

5.   To download a PDF file of the information you’ve viewed, click the Download icon on the upper right-hand corner.

B-Notice

The CP2100 Notice (B-Notice) from the IRS indicates that a payer may be responsible for backup withholding on the listed recipients that were previously submitted to the IRS. These listed recipients are included due to incorrect or missing name/TIN data. The IRS requires solicitation of the identified recipients to obtain correct information.

To enter B-Notice print information and submit it, perform the following steps:

1.   From the Navigation Menu, select Print and click the B-Notices card.

2.   Select the appropriate dates under the B-Notice Info section and enter the Return Info address that you want the B-Notice information send to.

a. Depending on which date is later, the Notice Date field should either be the date listed on the CP2100/CP2100A letter or the date that the letter was received by the payer.
b.   The Response Date field displays the desired date for the documents to be returned. The IRS states that this date should be 15 business days after the selected notice date since unresponsive recipients should be placed in back-up withholding 30 business days after the notice date.
3.   Click Next to view the next step.
4.   Under the 2018 Recipient Lookup search bar, you can search and add one or more recipients by name, TIN, or account. Hover over the TIN to view the full TIN for that recipient. The recipients entered in the B-Notice order should be those listed in the CP2100 letter. If a recipient’s TIN or name has been corrected or changed since the original submission, a solicitation may not be required.

Note: You cannot add a single recipient more than once. In addition, when clicking outside of the sliding panel, the information you have entered will not be lost.

5.   To delete one or more recipients from the B-Notice Recipient list, check the box associated with that user and click the icon.

a. A 1BN (First B-Notice) status should be assigned to individuals who appear on the CP2100/CP2100A letter for the first time and will receive a W9 form.
b.   A 2BN (Second B-Notice) status should be assigned to individuals who have appeared on another CP2100/CP2100A list within a three-year calendar period. The second B-Notice will not include a W9 form since it requires the recipient to validate their information from the SSA/IRS using their Social Security card or Letter 147-C.
6.   Click Next.
7.   Depending on the recipient(s) you’ve selected, the Order Info section will show how many recipients have 1B-Notices and how many have 2B-Notices. Click View Sample to view the 1B-Notice or 2B-Notice sample document.

8.   The Flat Fee and Print & Mailing Fee will be listed under Order Info.

9.   If you need to revise any information after reviewing the B-Notice Info and Return Info sections, navigate back to step 1, Setup B-Notice Order.
10.   If you want to cancel the B-Notice order, click Cancel. From the pop-up window, click Cancel Order.

11.   Click Submit to submit the B-Notice order.

W-9 Solicitations

Gathering the correct Tax Information Number (TIN) and Name from your recipients is essential for compliance purposes. TIR-Corporate’s W-9 Solicitations feature allows you to easily request the TINs and relevant certifications from your recipients. W-9 forms are typically sent after TIN matching to recipients with mismatched information as correcting these errors reduces possible penalties.

To enter W-9 solicitation print information and submit it, perform the following steps:

1.   From the Navigation Menu, select Print and click the W-9 Solicitations card.

2.   In the Return Info section, enter the address you would like the W-9 forms to be returned to.

3.   Click Next.
4.   Under W-9 Recipients, relevant recipients for the associated account will be auto-populated with the records that have been matched with the IRS. Using the search bar under the 2018 Recipient Lookup, you can add additional recipients searching by name, TIN, or account.

Note: You cannot add a single recipient more than once. In addition, when clicking outside of the sliding panel, the information you have entered will not be lost.

5.   To delete one or more recipients from the W-9 Recipient list, check the box associated with that user and click the icon.

6.   Click Next.
7.   Under Order Info, click View Sample to view the W-9 Solicitations sample document.

The Flat Fee and Print & Mailing Fee will be listed under Order Info.

8.   If you need to revise any information after reviewing the Order Info and Return Info sections, navigate back to step 1, Setup W-9 Order.
9.   If you want to cancel to W-9 order, click Cancel to open the pop-up window and click Cancel Order.

10.   Click Submit to submit the W-9 order.
