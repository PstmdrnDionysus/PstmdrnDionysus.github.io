Navigation

The Navigation Menu allows you to access the many aspects of the Sovos TIR-Corporate application, as well as the changing of contexts.

Context

To change the context, perform the following steps:

1.   Click on the blue or orange highlighted bar that will display either Prod or Test.

2.   The drop-down list will provide all contexts available.

3.   For each tax year, Prod and Test contexts will be available; contexts for the following tax year will be generated in April. The purpose of the Test Context is to test the loading of all data and resolving rejects. The purpose of the Prod context is to upload final data, correct errors, print, and transmit.

Note: You cannot view data loaded into a Test 2018 context in a Prod 2018 context or any other context.

Navigation Menu

The Navigation Menu allows you to navigate the many aspects of the application to perform for Tax Information Reporting purposes.

Overview

The Overview page contains key information about the current status of a your Tax Information Reporting. From this page, you can access the User Guide and the most recent Release Notes.

Data

The Data page is where the imports and manually added records are located. You can view key information about the state of your data, including the number of TIN mismatches.

Print

From the Print page, you can create and approve print files, as well as view previous prints and their current status.

Transmit

From the Transmit page, you can create and approve transmittal files, as well as view previous transmittals and their current status.

Reports

The Reports page contains various canned reports that can be output as an Excel file.

File Manager

The File Manager page contains all active files within the system. From this page, you can upload and download import, print, and transmittal files.

Settings

Click the Settings icon, , from the Navigation Menu to be brought to the Settings page. From this page, you can access user management and historic release notes.

Account Management

Click the Account Management icon, , from the Navigation Menu to view your Account Management information. You can access your account settings using My Account, and you can also logout. This information can be found at the bottom of the Navigation Menu.

Note: When you contact Sovos Support, a CID (Client ID) will be requested. The Version displays the last date the application was updated.
