Data Entry

The following section details the data entry process in TIR - Corporate.

Workflow

The order of operations for entering data into TIR - Corporate is as follows:

1.   Manually enter or import Payer File(s).
2.   Import Recipient File(s).
3.   Import Form File(s).

Data Entry Process

To manually enter data, perform the following steps:

1.   From the Navigation Menu, select Data.

2.   Click Add Record on the upper right-hand corner of the page.

3.   Select the appropriate record type from the associated drop-down list. The following options are provided: Payer, Recipient, 1042-S Recipient, 1042-S, 1099-DIV, 1099-INT, and 1099-MISC.

Payer

1.   When you select Payer from the drop-down list, the Add Payer sliding panel will be displayed with the following fields.

2.   The following fields in red text are required to be populated to save a payer.

3.   You can populate Payer ID with any value that is able to be tied back to each individual form. When mapping a form import, PTIN will be used to represent the Payer ID; the value entered in the Payer ID field must be the value entered in the PTIN field on a form import.

Note: US is not an option in the Country Code drop-down list. If the Country Code should be US, leave this field blank.

4.   If the payer you are entering has any state IDs, enter the State and State ID value. If you need to enter additional state and state ID values, click Add Another State to add another row.

5.   Once you have entered all required and relevant information, click Save.

Recipient

1.   When you select Recipient from the drop-down list, the Add Recipient sliding panel will be displayed with the following fields.

2.   The following fields in red text are required to be populated to save a Recipient.

3.   To perform a real-time TIN match, you must enter a TIN and Name; see TIN Matching for more information.
4.   Once you have entered all required and relevant information, click Save.

Forms

1.   When you select the 1099-DIV, 1099-INT, or 1099-MISC form from the drop-down list, the associated sliding panel with the following information will be displayed.

2.   Forms must have a payer and recipient attached. To add Payer Information, click Add and enter the relevant Name or ID.

3.   To add Recipient Information, click Add and enter the relevant Name, TIN or Account.

Note: To search using a TIN, you must enter the full TIN. For the Name and Account Number, you must enter at least three characters.

4.   To change payer or recipient information, click the associated Change button.

5.   Once you have add a recipient, the Recipient tab will appear in the banner.

6.   To delete a form, recipient, or payer, click the icon to the left of the Cancel button and click Delete.
